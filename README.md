# Student-Marksheet-Sample-Data
An Excel-based student performance dashboard that analyzes marks, grades, and pass/fail status using formulas, charts, and data visualization.
# Objective
To analyze student academic performance and present it in a clear and interactive dashboard for better understanding and evaluation.
# Dataset Details
The dataset includes:
Student ID , 
Student Name , 
Marks in Math, Physics, and Chemistry ,
Total Marks ,
Percentage (%) ,
Grade (A, B, C, etc.) , 
Pass/Fail Status
# Dashboard Insights
The dashboard provides:
Total number of students , 
Average percentage , 
Grade-wise student count , 
Pass and fail count , 
Performance distribution using charts 
# Features
Automatic grade calculation using formulas , 
Conditional formatting for better visualization , 
Summary box for quick insights , 
Grade distribution chart , 
Clean and structured data layout ,
